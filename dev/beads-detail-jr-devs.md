# Review Beads
Used for reviewing beads and ensuring that they are detailed and need no additional clarification and detail before development begins

```
Carefully and with extreme detail and ruthlessness, review all of the beads for this project and ensure that they detailed and clear enough so that the only information that the developers need to implement the bead is the content of the bead and the codebase itself

Keep in mind that junior developers will be working on each bead so it needs to be crystal clear on what to do within each bead
```
