# URL Patterns

Using a list of URLs from SF (or really any source), have an LLM identify patterns from the uploaded list. 

_Note: This is for web versions of LLMs and require a CSV file_

```
Attached is a CSV list of URLs for {{website}} which does {{description of website and products/sercices they offer}}

I want you to carefully and with extreme detail review each row line by line to identify common URL patterns that emerge across all of the URLs. Be extremely meticulous, detail oriented, and comprehensive with your analysis. Provide both a CSV and word/MD document from your analysis.

Provide both a CSV and MD document from your analysis.

The CSV should contain a list of all the URLs along with the pattern(s) that it matches

The MD document should provide a break down of each pattern type and explanation behind it
```
